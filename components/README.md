# Components

React components organized by functionality.

## Structure

- `ui/` - Base Chakra UI components (wrappers and customizations)
- `forms/` - Form components using React Hook Form + Zod
- `charts/` - Chart components for portfolio visualization
- `tables/` - Data tables for positions and transactions
- `fifo/` - FIFO-specific visualization components
- `layout/` - Layout components (Navbar, Sidebar, Footer)
- `admin/` - Admin panel components
